# OLED Viewing Angle Analyzer - Project Rules

## Project Overview
- 이 프로젝트는 **매거진 형태**로 운영되는 OLED 시야각 분석 도구입니다.
- GitHub Pages로 배포: https://waterfirst.github.io/oled-viewing-angle/
- 저자: Nakcho Choi (Samsung Display)

## Important Rules

### 1. 개인 연금 콘텐츠 금지
- **개인 연금(pension) 관련 콘텐츠는 이 프로젝트에 포함하지 않는다.**
- pension_dashboard.html 등 연금 관련 파일은 생성하거나 커밋하지 않는다.
- 연금, 퇴직금, 재무설계 등 개인 재정 관련 내용은 이 저장소에 넣지 않는다.

### 2. 키메라(Chimera) 분석 보고서
- 키메라 AI 분석 보고서는 별도 저장소에서 관리: https://waterfirst.github.io/chimera-ai/
- **매주 키메라 분석 보고서를 업로드하고 목차에 반영해야 한다.**
- chimera-ai 저장소: https://github.com/waterfirst/chimera-ai

### 3. 매거진 운영 원칙
- 콘텐츠는 매거진 형식에 맞게 정리하여 게시한다.
- 각 콘텐츠는 독립적인 HTML 페이지로 제작한다.
- 시각적 완성도와 데이터 인터랙션을 중시한다.

## Repository Structure
- `index.html` - 메인 OLED 시야각 분석기 (v2.0)
- `data.csv` / `data_clean.csv` - 측정 데이터
- `CLAUDE.md` - 프로젝트 규칙 (이 파일)
