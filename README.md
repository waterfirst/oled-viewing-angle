# OLED Viewing Angle Analyzer

[![Demo](https://img.shields.io/badge/Demo-Live-brightgreen)](https://waterfirst.github.io/oled-viewing-angle/)

**[▶ Launch Analyzer](https://waterfirst.github.io/oled-viewing-angle/)**

Interactive web tool for analyzing OLED display viewing angle performance — color shift (Δu'v'), luminance roll-off, and CIE chromaticity tracking.

## Features
- 6 interactive Plotly charts
- CSV upload (phi, theta, Y, x, y) or drag-and-drop
- Built-in sample data generator
- CIE xy → u'v' automatic conversion
- Spec compliance check (Δu'v' @ 45°/60°, Y ratio)
- Polar map of color shift
- Dark/Light theme, EN/KR language toggle
- CSV export of processed data

## Usage
1. Click "Load Sample Data" for a quick demo
2. Or upload your measurement CSV (columns: phi, theta, Y, x, y)
3. Select azimuth angle(s) to analyze
4. Check spec compliance in sidebar
